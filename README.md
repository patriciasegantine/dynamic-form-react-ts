# Dynamic Form

## Practicing how to use a dynamic form, using:

- Antd Design
- Fake API (json server)
- Imask
- Phosphor React Icon
- React Router
- Redux
- Styled Component
- Typescript
- Vite

## How to use:

- npm install - to install all dependencies
- npm run server - to start the jason server
- npm run dev - to start the application
